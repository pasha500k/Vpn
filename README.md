# Launchpad VPN Platform

Launchpad VPN — это комплекс из VPN-сервера, веб-панели и настольного клиента с двумя режимами работы: туннелирование всего трафика или только выбранных доменов. Проект написан на Python и использует общую SQLite-базу для ключей, пользователей и сессий.

## Структура

- `server/` — асинхронный TLS-прокси, реализующий VMess-совместное рукопожатие, контроль сессий и квоты по режимам.
- `client/` — кроссплатформенный VMess-клиент, который можно собрать в `.exe` и который поддерживает вход по аккаунту или ключу.
- `webapp/` — FastAPI-портал с главной страницей для пользователей и закрытой админ-панелью управления ключами и доменами.
- `config/` — файлы конфигурации для сервиса.
- `pyproject.toml` — список зависимостей.

## Возможности

- Два режима туннеля: «Весь трафик» (по умолчанию 5 часов в сутки без ключа) и «Домены» (1 час в сутки без ключа).
- Активационные ключи (UUID), которые снимают ограничения по времени и дают безлимит на оба режима.
- Учётные записи пользователей с безопасным хэшированием паролей и ведением журнала VPN-сессий.
- Настройка списка разрешённых доменов для доменного режима прямо из панели.
- TLS-шифрование между клиентом и сервером и VMess-совместимый протокол обмена.
- Веб-интерфейс с главной страницей для регистрации и загрузки Windows-клиента и защищённая админ-панель.

## Быстрый старт

### 1. Подготовка окружения

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Для Windows-клиента используйте `pyinstaller client/vpn_client.py --onefile` для упаковки в `.exe`.

### 2. Инициализация базы данных

```bash
python webapp/manage.py migrate
```

### 3. Настройка администратора и запуск веб-интерфейса

```bash
uvicorn webapp.main:app --reload --host 0.0.0.0 --port 8000
```

> По умолчанию активен администратор `pasha500k` / `Hehetoto123`. Эти значения можно переопределить переменными `ADMIN_USERNAME` и `ADMIN_PASSWORD` **до** запуска, чтобы сменить пароль.

### 4. Запуск VPN-сервера

```bash
python server/vpn_server.py
```

### 5. Подключение VMess-клиента

```bash
python client/vpn_client.py \
    --server vpn.aitelegrama.com \
    --port 9443 \
    --mode full \
    --username demo-user \
    --password secret123 \
    --domain example.com \
    --target-port 443 \
    --payload-file request.txt \
    --ca certs/server.pem
```

Либо укажите `--key <UUID>`, чтобы воспользоваться активационным ключом и получить безлимитный доступ. Клиент устанавливает TLS-сессию, затем выполняет VMess-рукопожатие с выбранным режимом. В ответ сервер сообщает идентификатор, список разрешённых доменов (для режима «Домены») и оставшееся время сессии.

### 6. Настройка домена

Настройте DNS-запись типа `A` или `CNAME` на ваш сервер, например `vpn.aitelegrama.com`.

## Дополнительно

- В `webapp/templates/home.html` настраивается главная страница для пользователей с кнопкой скачивания готового `.exe` и формами регистрации/активации.
- Суточные квоты (5 часов для «Весь трафик» и 1 час для «Домены») задаются в `shared/services.MODE_LIMITS_SECONDS` и применяются ко всем пользователям без ключей.
- Конфигурацию доступных доменов можно заполнить в `config/domains.yml` или через админ-панель (`/admin`).
